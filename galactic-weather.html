Galactic Weather App

UI Design

This was a project for my UI design class. The assignment was to design an interface for a mobile app that a galactic traveler would use while preparing for a voyage.

The user would have to be able to query the weather at any of the 9 planets in order to prepare their route. They would also have to get alerts about any major galactic weather events so as to plan their voyage accordingly.

I had a lot of fun designing this one! I went for a futuristic space-age feel. On my home screen, I used a circular carousel to imitate the planets revolving around the sun.

When you click on a planet, it takes you to the weather report which includes the forecast for the upcoming week as well as any major weather events. You can then slide through the planets at the top to easily navigate between them. You can also click the + next to a planet's name to add it to a custom screen for easy comparison between planets.

Web Design

Overview

This was my first attempt at building a personal website from scratch. I'd been fascinated by the Parallax trend, particularly by its use in the Firewatch website, so I decided to create something similar of my own.

The flat landscape header image was a lot of fun to make. I drew each layer individually in Photoshop and then animated each with a different speed to create the parallax effect.

I also had a lot of fun experimenting with different colors and effects (the link effects are a personal favorite of mine).

I ultimately replaced the website with my current one both in order to create a more portfolio-based showcase of my work and to speed up loading time, but I had a lot of fun building that site and learned a lot through the process.
